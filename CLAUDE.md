# Meetings Countdown Pro

macOS menu-bar app (Python + PyQt6) that plays a broadcast-style countdown before calendar meetings.

## Key References

- **Product spec:** read [SPEC.md](./SPEC.md) before making feature decisions
- **Testing plan:** [TESTING.md](./TESTING.md)

## Dev Setup

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python -m meetings_countdown_pro
```

Requires Python 3.11+ (python.org framework build, not Homebrew).
